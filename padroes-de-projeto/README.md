## Sistema de Gerenciamento de Aviação — Padrões de Projeto

Este projeto consiste em uma aplicação simplificada em Java para o domínio de **Aviação**, focada no gerenciamento de voos. O objetivo principal é demonstrar a aplicação prática de padrões de projeto (Design Patterns) criacionais, estruturais e comportamentais.
